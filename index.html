Giphy Producer

With the Giphy producer page, a user is able to use a created button which returns the top 10 gifs for that search term calling from the Giphy API.

If the user instead has a different term to search for, users can enter their term in the input field and have a button dynamically created for them. They then can click that button to perform the API call to return the top 10 gifs related to the search term that they input.

Once the gifs are created and placed into the DOM, the user can then start or stop the gif simply by clicking on it.

Film Trivia!

A fun trivia game that utilizes Giphy API, jQuery, Bootstrap and Google Fonts.

This game coincided with learning Bootstrap and jQuery. To start, click the start button. You will have 15 seconds for each of 10 questions. To answer, click on the answer. A fun gif that pertains to the question will play in between each answer. At the end you will be shown the amount of answers that you got right, wrong, and those that went unanswered. Learn Movie Trivia!
